# Chef à la Maison 🍽️

Chef à la Maison est une application web interactive de recettes de cuisine.  
Elle permet aux utilisateurs de découvrir, enregistrer et partager des recettes dans une interface moderne et intuitive.

Le projet a été développé avec **HTML, CSS et JavaScript** sans utiliser de framework.

---

## 🎯 Objectif du projet

L'objectif est de créer une application web interactive où les utilisateurs peuvent :

- découvrir des recettes de cuisine
- consulter les ingrédients et les étapes de préparation
- regarder des vidéos explicatives
- enregistrer leurs recettes préférées
- interagir avec les recettes grâce aux notes et commentaires

Ce projet permet de mettre en pratique les bases du développement web et la manipulation du DOM avec JavaScript.

---

## ✨ Fonctionnalités

### 📖 Catalogue de recettes

Les recettes sont affichées sous forme de cartes avec :

- image
- titre
- catégorie
- niveau de difficulté

### 🔎 Recherche et filtres

Les utilisateurs peuvent :

- rechercher une recette par son nom
- filtrer les recettes par catégorie

Catégories :

- Entrées
- Plats
- Desserts
- Boissons

### 👤 Compte utilisateur

Les utilisateurs peuvent :

- créer un compte
- se connecter
- interagir avec les recettes

### ❤️ Favoris

Les utilisateurs connectés peuvent ajouter des recettes à leurs favoris afin de les retrouver facilement.

### ⭐ Notes et commentaires

Chaque utilisateur peut :

- noter une recette sur 5 étoiles
- laisser un commentaire

### 🎥 Vidéos YouTube

Chaque recette peut contenir une **vidéo YouTube intégrée** afin de suivre la préparation étape par étape.

### 🛠️ Mode administrateur

Un administrateur peut :

- ajouter une recette
- modifier une recette
- supprimer une recette
- gérer les vidéos

### 🌙 Mode clair / sombre

Un bouton permet de basculer entre :

- mode clair
- mode sombre

### 📱 Design responsive

L'application s'adapte automatiquement aux :

- ordinateurs
- tablettes
- smartphones
