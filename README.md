# Chef à la Maison

## Description

Chef à la Maison est un site web interactif de recettes de cuisine. L’utilisateur peut découvrir des recettes, les ajouter à ses favoris et même créer ses propres recettes. Le site est entièrement statique et développé avec HTML, CSS et JavaScript.

## Fonctionnalités

- Galerie de recettes avec images et descriptions
- Ajout de nouvelles recettes par l’utilisateur
- Sauvegarde des recettes favorites via localStorage
- Filtre par catégorie (Entrée, Plat, Dessert)
- Interface responsive et design moderne

## Installation / Utilisation

1. Télécharger ou cloner le projet.
2. Ouvrir le fichier `index.html` dans un navigateur web.
3. Interagir avec les recettes, ajouter vos propres recettes et les sauvegarder.

## Technologies utilisées

- HTML5
- CSS3
- JavaScript (ES6)
- LocalStorage pour la sauvegarde des favoris
