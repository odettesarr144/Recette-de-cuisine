# Spécifications du projet - Chef à la Maison

## Objectif

Développer un site web statique en HTML, CSS et JavaScript permettant aux utilisateurs de découvrir, ajouter et sauvegarder des recettes de cuisine.

## Fonctionnalités principales

1. Affichage d’une galerie de recettes avec image, titre et description.
2. Possibilité pour l’utilisateur d’ajouter ses propres recettes via un formulaire.
3. Fonctionnalité “Favoris” permettant de sauvegarder des recettes dans le navigateur (localStorage).
4. Filtrage des recettes par catégorie (Entrée, Plat, Dessert).
5. Interface moderne et responsive adaptée aux mobiles et tablettes.
6. Hero section attractive pour accueillir l’utilisateur.
7. Design inspiré de Netflix / Apple : cartes interactives, hover animations, et style épuré.

## Structure technique

- HTML : structure du site et formulaires
- CSS : design et mise en page
- JavaScript : gestion dynamique des recettes, ajout aux favoris et affichage interactif.

## Contraintes

- Projet 100% statique (aucun serveur requis)
- Compatible tous navigateurs récents
- Responsive design obligatoire
